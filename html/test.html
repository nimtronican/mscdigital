Data tables

Data tables provide a way to organize and clearly present relational information. With data tables, you can alternate columns or rows, add color to cells or create a grid-style format. With the data table widget, you can add advanced functionality to your data tables.

  • All data tables require the class ibm-data-table for proper default styling.
  • Do not use tables for layout purposes; instead, use grids. Tables should be used only to display relational content or data.
  • Use the ibm-hidden-small class to set the element to display:none on smaller screens.

    • data-paging="true" (enables pagination)

Data table – horizontal scroll

The horizontal scroll feature allows your table to expand beyond the size of the column and scroll horizontally. This is useful when you have a wide table with several columns, but want to constrain it to a limited display area. To enable horizontal scrolling, add the data-scrollaxis="x" attribute to your <table>.
